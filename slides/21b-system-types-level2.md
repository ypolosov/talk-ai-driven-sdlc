# Аналогия: Духовка vs Оператор tenant

<v-clicks>

| Вид системы | Выпечка | iGaming |
|---|---|---|
| **Целевая система** | Духовка | Оператор tenant |
| **Надсистема** | Пекарня | Бизнес-операции оператора |
| **Системы в окружении** | Холодильник, миксер, касса | KYC, payment gateway, CRM |
| **Подсистемы** | ТЭНы, терморегулятор, камера | Tenant config, game catalog, DB |
| **Система создания** | Фабрика + конвейер | iGaming Platform |

</v-clicks>
