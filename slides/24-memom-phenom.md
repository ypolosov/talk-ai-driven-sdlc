# Мемом vs феном

<v-clicks>

И тут нам поможет пара понятий из системного мышления:

**Мемом** - описание, замысел. 

**Феном** - воплощение, реализация.

| Мемом (описание) | → | Феном (воплощение) |
|---|---|---|
| Requirements / User Stories | → | **Реализованное поведение** системы |
| ADR (Architecture Decision Record) | → | **Архитектура работающей системы** (реализованные решения в коде и инфраструктуре) |
| Test spec | → | **Прошедший тест** (зелёный результат) |
| Код (исходный) | → | **Работающий продукт** (deployed, running system) |

</v-clicks>
