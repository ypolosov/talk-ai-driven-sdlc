- а что меняется когда, цепочка создания ценности меняется вот так?
  - компании которая специализируется на производстве и продаже промышленного пекаренного оборудования, построила фабрику-лабораторию с тестовым конвеером где оттачивается производство духовок, и построили лабораторию-пекарню где на этих духовках оттачивают выпикание пирожков, которые продают прохожим
  - причины этого обычно:
    - Dogfooding — компания-производитель оборудования сама использует свои духовки в лаборатории-пекарне для выпекания пирожков и продажи прохожим
    - Reference Implementation — лаборатория-пекарня с тестовым конвейером и духовками служит эталонным примером для потенциальных клиентов (хозяев фабрик): "вот так выглядит работающая пекарня на нашем оборудовании, вот качество пирожков".
    - Captive operator — эта лаборатория-пекарня принадлежит самой компании-производителю оборудования, а не внешнему клиенту-пекарю. Компания сама выступает в роли своего клиента — владельца фабрики и пекаря одновременно.
